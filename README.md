# ReddIF API

## 🚀 Como rodar o projeto

### 1. Pré-requisitos

* .NET SDK instalado (recomendado: **.NET 10**)
---

### 2. Acesse a pasta do projeto

No terminal, vá até a pasta onde está o arquivo `.csproj`:

```bash
cd ReddIF/ReddIF
```

---

### 3. Execute o projeto

```bash
dotnet run
```

---

### 4. Acesse a API

Após rodar, você verá algo como:

```
Now listening on: http://localhost:xxx
```

Abra no navegador:

```
http://localhost:xxxx/scalar/v1
```

---

📌 Observações
A interface Scalar será aberta para testar os endpoints da API.
Caso haja erro, verifique se a versão do .NET está compatível com o projeto.
